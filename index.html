VS Code

  • To comment, use '/*' before and '*/' after.

HTML

  • The head element contains information about the webpage.
  • The 'title' element contains the title which is good for SEO and appears as the tab name.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • .card is the value of 'class', indicating the specific sections we want the CSS style to apply to.
  • # is used before id selectors, which apply CSS effects on more specific sections.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • Use 'function "function name" () { code to execute } as formula.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Array: a single variable used to hold a group of data. Can contain different kinds of data.
  • String: ?
  • For loop statements
  • Starting point for loop. e.g. var x=0
  • Limit/condition. e.g. x < shapes.length>
  • Pattern? e.g. x++
